# equipment_rental_task



Maqsad Uskunalarni ijaraga berish uchun oddiy boshqaruv tizimi yaratish.

Functional Requirements

1.  Equipment Management

Admin quyidagilarni qila olishi kerak:

Equipment CRUD - uskuna qo‘shish - edit qilish - delete - ro‘yxatni
ko‘rish

Har bir uskuna: - id - name - category - daily_price - status -
available - rented - maintenance - image (optional)

2.  Rental Management

Yangi ijara yaratish

Form: - client_name - phone - equipment - start_date - end_date

System hisoblasin: - total_days - total_price

3.  Rental List

Ko‘rsatilsin: - client - equipment - rental dates - total amount -
status

Status: - active - returned - late

4.  Return Equipment

Qaytarilganda: - rental status → returned - equipment status → available
